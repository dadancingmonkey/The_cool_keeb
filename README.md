The CoolKeeb

Description
The CoolKeeb is a cool custom keyboard that was fully designed and made by me. 
It is a general gaming keyboard that has a premium enclosure and bluetooth wireless cabilities whille retaining RGB illumination. 

It includes:
- custom PCB that hosues the switches
- Hotswappable sockets 
- exploded 75% format with 80 keys + a volume knob
- Nice!Nano MCU 
- 2000mAh battery 
- Cherry MX Brown switches
- Per-Key RGB illumination
- side printed PBT keycaps
- 3d printed & aluminum sheetmetal enclosure
